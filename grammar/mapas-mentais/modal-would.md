# WOULD — Mapa Mental

```mermaid
mindmap
  root((WOULD))
    Condicional
      tradução: iria, faria
      tem condição implícita
      I would go if I had time
    Hábito passado
      tradução: ia, fazia
      sem condição implícita
      We would play outside every evening
      Antonio's grandfather would go to the Count's house
    Pedido educado
      tradução: você poderia
      Would you help me?
    would have
      tradução: teria
      intenção frustrada
        I would have called, but I lost my phone
```

## Resumo
| Uso | Tradução PT | Exemplo |
|---|---|---|
| Condicional | iria, faria | *I would help if I could* |
| Hábito passado | ia, fazia (imperfeito) | *She would sing every morning* |
| Pedido educado | você poderia | *Would you mind waiting?* |
| would have | teria | *I would have stayed* |

## ia vs iria — a armadilha clássica

> *He would go to the house every Sunday.*
> Ele **ia** à casa todo domingo. ✅ ← hábito → imperfeito
> Ele **iria** à casa todo domingo. ❌ ← soa condicional, como se faltasse "se..."

**Regra:** sem condição explícita ou implícita → imperfeito ("ia", "fazia", "comia")

## Não confunda
- **would** vs **used to** → hábito passado
  > *I would run every morning.* ✅ → ação repetida
  > *I used to run every morning.* ✅ → ação repetida
  > *I used to live in Porto Alegre.* ✅ → estado
  > *I would live in Porto Alegre.* ❌ → would não funciona para estados

- **would have** vs **could have**
  > *I would have gone.* → era minha intenção, mas algo impediu
  > *I could have gone.* → eu tinha como, mas não fui
